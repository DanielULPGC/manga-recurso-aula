# Informes

Carpeta destinada a informes de revision, auditorias, comprobaciones y registros
de estado del recurso.

## Informes disponibles

- `auditoria-inicial.md`: diagnostico tecnico inicial del repositorio.
- `auditoria-recursos.md`: referencias rotas, duplicados y recursos pesados.
- `cambios-recursos.md`: correcciones seguras aplicadas sobre recursos.
- `accesibilidad-basica.md`: mejora verificable de teclado, foco, ARIA y modales.
- `auditoria-seguridad-ia.md`: revision de HTML dinamico, sanitizacion e IA.
- `rendimiento.md`: auditoria de carga frontend y recomendaciones.
- `catalogo-validacion.md`: resultado de validacion formal del catalogo.
- `modularizacion-1.md`: primera fase de extraccion de constantes y utilidades.
- `materiales-docentes.md`: revision de PDF y materiales docentes exportables.
- `verificacion-pwa-2026-07-13.md`: verificacion PWA por HTTP local y Chrome headless.
- `estado-proyecto-2026-07-13.md`: registro de continuidad del proyecto.
